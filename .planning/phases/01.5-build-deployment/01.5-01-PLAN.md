<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| build config → bundle.js | Webpack config controls transformation of source code into production output |
| externals → runtime globals | Externalized modules trust Splunk runtime to provide React/ReactDOM globals |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-1.5-01 | Spoofing | externals mapping | mitigate | Use exact module names ('react'/'react-dom') — no wildcard patterns that could map to wrong globals |
| T-1.5-02 | Tampering | Terser config | accept | TerserPlugin with drop_console only removes console.* calls; minification is deterministic, no inject risk |
| T-1.5-03 | Information Disclosure | devtool sourcemaps | mitigate | Production build sets devtool: false — no source code leaks in production bundle output |
</threat_model>
